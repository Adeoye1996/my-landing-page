Maze Solver

Creating a Game in 3D using raycasting. Designed for kids so they can improve their spatial recognition and analytical skills

Try the Maze Solver App

Key Features

Customizable Maze Generation

Create unique mazes of varying sizes and complexities.

Multiple Solving Algorithms

Solve mazes using different algorithms such as DFS, BFS, and A*.

Real-Time Solving Visualizations

Watch as the algorithm finds the solution in real-time.
